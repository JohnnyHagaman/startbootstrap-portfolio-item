Climbing Rating Classifier Predicting Moonboard Technical Grades

Project Description

Amongst the climbing community technical climbing grades are infamously known to be subjective and often inaccurate. This project attempts to predict Vermin scale equivalent climbing grades by modeling bouldering problems submitted to the Moonboard climbing problem database, a uniform crowdsourced training board utilized across the world.

    Github Repository

Project Details

  • Selenium scraper for Moonboard.com
  • Extensive feature engineering with k-d trees
  • Models: Linear, Boosted CART, Neural Network
  • Multiple model evaluation metrics
